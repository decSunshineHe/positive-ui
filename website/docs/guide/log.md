# 版本发布

当前稳定版本：`0.1.2`

当前测试版本：`0.1.3-beta.2`

## 稳定版本

beta 版本经历过内部提测后会发布稳定版本

## 测试版本

版本发布通常会经历不定量的 beta 版。
测试版本是为了进行集成/稳定性测试，并让早期用户为不稳定的功能提供反馈。请不要在生产中使用测试版本。所有的测试版本都被认为是不稳定的，并且可能会在两者之间产生不兼容变更。
